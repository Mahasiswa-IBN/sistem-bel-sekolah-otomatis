Ikuti langkah-langkah di bawah untuk merakit hardware dan mengunggah program ke mikrokontroler Anda.

1. Skema Kabel NodeMCU ESP8266
Info Pin Bebas Konflik:

Secara default, pin I2C untuk RTC DS3231 berada di D1 (SCL) dan D2 (SDA). Oleh karena itu, jalur komunikasi ke DFPlayer Mini telah disesuaikan ke pin D5 (RX) dan D6 (TX), serta pemicu Relay ke pin D7 guna menghindari bentrokan sinyal data.

Perangkat Pendukung Pin Perangkat Pin NodeMCU ESP8266 Keterangan
DS3231 RTC VCC 3V3 Catu daya RTC
GND GND Ground
SDA D2 (GPIO 4) Jalur data komunikasi I2C
SCL D1 (GPIO 5) Jalur clock komunikasi I2C
DFPlayer Mini VCC VIN / 5V Catu daya Audio (Ambil dari daya USB 5V)
GND GND Ground
RX D6 (GPIO 12) Wajib gunakan Resistor 1K Ohm secara seri ke pin RX DFPlayer
TX D5 (GPIO 14) Jalur kirim audio dari DFPlayer ke NodeMCU
Relay Modul (Opsional) VCC VIN / 3V3 Daya kumparan relay
GND GND Ground
IN (Signal) D7 (GPIO 13) Pin Kontrol (Relay aktif otomatis saat bel berbunyi)
PERHATIAN: Resistor 1K Ohm

Anda Wajib menyolder sebuah Resistor 1K Ohm di antara kabel Pin **D6 NodeMCU** ke pin **RX DFPlayer Mini**. Resistor ini berfungsi untuk meredam noise (dengung kencang) pada speaker dan melindungi modul DFPlayer Mini agar awet.

2. Struktur Folder & Lagu pada MicroSD Card

Modul pemutar MP3 DFPlayer Mini memerlukan format penyimpanan kartu SD yang khusus agar lagu tidak tertukar dan dapat diputar secara presisi:

  • 1
    Format SD Card:

    Format kartu memori MicroSD Anda dengan file sistem FAT32 (gunakan kapasitas maksimal 16GB atau 32GB).

  • 2
    Buat Folder "01":

    Buatlah sebuah folder baru di dalam MicroSD Anda dengan nama **`01`** (harus terdiri dari dua karakter angka tersebut).

  • 3
    Penamaan File Lagu (3 Digit):

    Simpan lagu bel sekolah Anda ke dalam folder `01` dan namai lagunya dengan awalan 3 digit angka seperti:

    /01/001.mp3 -> (Bel Masuk Kelas)
    /01/002.mp3 -> (Bel Pergantian Jam)
    /01/003.mp3 -> (Bel Istirahat)
    /01/004.mp3 -> (Bel Pulang Sekolah)

    Pada dashboard web jadwal bel, isikan Folder MP3: 1 dan File MP3: 1 untuk memicu file `001.mp3` di folder `01`.

3. Langkah Pemrograman & Upload LittleFS
A. Cara Upload Kode Arduino
  1. Buka aplikasi **Arduino IDE** pada komputer Anda.
  2. Instal Library pendukung melalui menu **Tools > Manage Libraries...**:
    • ArduinoJson (Oleh Benoit Blanchon)
    • RTClib (Oleh Adafruit)
    • DFRobotDFPlayerMini (Oleh DFRobot)
  3. Buka berkas program Arduino:
    • Buka **`bel_sekolah_nodemcu/bel_sekolah_nodemcu.ino`** (Disarankan: Khusus untuk NodeMCU ESP8266).
    • Atau buka **`bel_sekolah/bel_sekolah.ino`** (Mendukung ESP32 & ESP8266 sekaligus).
  4. Pilih board **NodeMCU 1.0 (ESP-12E Module)** di menu *Tools > Board*.
  5. Hubungkan NodeMCU ke PC melalui USB, tentukan port COM yang aktif di menu *Tools > Port*, kemudian tekan tombol **Upload** (panah kanan).
B. Cara Upload index.html ke Memori NodeMCU

Agar NodeMCU dapat menampilkan website saat diakses, file `index.html` harus di-upload ke memori flash internal (LittleFS):

  1. Pastikan Anda telah memasang plugin **ESP8266 LittleFS Data Upload Tool** pada Arduino IDE.
  2. Di dalam folder sketch pilihan Anda (`bel_sekolah_nodemcu` atau `bel_sekolah`), buat folder baru bernama **`data`**.
  3. Salin berkas **`index.html`** dari projek Anda ke dalam folder `data` tersebut (sehingga strukturnya menjadi `bel_sekolah_nodemcu/data/index.html` atau `bel_sekolah/data/index.html`).
  4. Tutup **Serial Monitor** pada Arduino IDE agar port COM tidak sibuk.
  5. Di Arduino IDE, klik menu **Tools > ESP8266 LittleFS Data Upload**.
  6. Tunggu hingga proses selesai tertulis *"LittleFS Image Uploaded"*.
